Per RSR policy, implementation will use:

[cols="1,2"]
|===
|Category |Allowed Technologies

|Runtime |Deno
|Package Management |Guix (primary), Nix (fallback)
|Configuration |Nickel, Guile Scheme
|Scripting |Bash/POSIX (minimal)
|===
